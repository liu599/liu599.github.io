# liu599.github.io

个人 GitHub **Star 管理**仓库（兼 GitHub Pages 站点）。

用来整理、浏览我 star 过的仓库，而不是放项目代码。

## 在线页面

- 首页：https://liu599.github.io/
- Stars 整理：https://liu599.github.io/stars.html

## Stars 整理页

[`stars.html`](stars.html) 整理全部 star，支持搜索、筛选和数量分布。

当前分类方式：

1. **兴趣主题**（优先）——游戏技术/休闲、装机/系统软件/工具、学习/书/资源、AI 应用、传统机器学习深度学习/AI 标注·训练·推理·数据、网络代理/爬虫/安全、测试、界面技术（Web/Terminal/端侧 App UI）、后端、多媒体；拿不准的放「争议」
2. **主要语言**——其余按语言归组（C/C++/C# 已合并）




之后会继续加主题类、再细分。

## 仓库内容

| 文件 | 说明 |
|------|------|
| `stars.html` | Star 列表整理页（数据内联） |
| `index.html` | 个人主页，含整理页入口 |
| `stylesheets/` | 站点样式 |

数据来自 GitHub API 的用户 star 列表，页面为静态 HTML，无后端。
